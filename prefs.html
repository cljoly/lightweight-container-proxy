Fallback proxy

Used as a fallback when no options are defined. With option direct, no proxy is used and option socks define a proxy to use.

Proxy per container

For each container, requests will be proxied (socks option), sent directly (direct option) or fallback settings from the above section will be used (fallback option).

Proxy in incognito mode

Set proxy used in incognito mode aka private browsing. Note that the extension needs to be allowed in private mode for this to have any effect.
